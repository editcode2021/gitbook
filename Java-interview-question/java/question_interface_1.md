# 基础概念题

下面哪一项说法是正确的

    1. 在一个子类里,一个方法不是 public 就不能重载
    2. 覆盖一个方法只需要满足相同的方法名和参数类型
    3. 覆盖一个方法必须方法名,参数和返回类型都相同
    4. 一个覆盖的方法必须有相同的方法名,参数名和参数类型

答案 3

覆盖函数与被覆盖函数只有函数体不同

下面哪一项说法是错误的

    1. 重载函数的函数名必须相同
    2. 重载函数必须在参数个数或类型上有所不同
    3. 重载函数的返回值必须相同
    4. 重载函数的函数体可以不同

答案 3

函数的重载与函数的返回值无关

下面哪一项说法是正确的

    1. 静态方法不能被覆盖
    2. 静态方法不能被声明称私有
    3. 私有方法不能被重载
